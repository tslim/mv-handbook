# Mindvalley Tech Handbook

The Mindvalley Tech handbook is the central repository for how we run the Tech team. Please make a pull request to suggest improvements or add clarifications. Please use issues to ask questions.

## Motivation

* Unify the teams with similar tools (e.g. Jira)
* Advocate good engineering practices across all teams
* A space for every team members to suggest improvements and give feedback

## Tools

* [Slack](mindvalley.slack.com): Communication
  * [#general](https://mindvalley.slack.com/archives/C02SYJ9N5) - everyone in MV includes contractors
  * [#mindvalleyteam](https://mindvalley.slack.com/archives/G0EELAUAE) - MV employees only
  * [#tech-general](https://mindvalley.slack.com/archives/G0C5M5A3S) - tech team only
  * [#tech-learning-circle](https://mindvalley.slack.com/archives/C0FCU4HFW) - tech team shares learning resources
  * [#updates-from-founder](https://mindvalley.slack.com/archives/CDA4CEEAK) - updates from CEO
* [Jira](https://mindvalley.atlassian.net/secure/Dashboard.jspa): Product management
* [Wrike](https://www.wrike.com/open.htm?id=296484967): Product management (replaced by Jira)
* [Github](https://github.com/mindvalley/): Source code management
